# 🎉 Upcoming Event Page

A responsive and user-friendly Event Management Website built using HTML, CSS, and JavaScript. This project allows users to explore upcoming events, view event details, browse galleries, and access contact information through an attractive and modern interface.

## 🌐 Live Demo

🔗 https://atchayaprojects.neocities.org/Event/event

## 📌 Features

- 📅 Display upcoming events
- 🎨 Modern and responsive UI design
- 🖼️ Event gallery section
- 📱 Mobile-friendly layout
- 📞 Contact information section
- ⚡ Fast-loading static website
- 🎯 Clean and organized code structure
